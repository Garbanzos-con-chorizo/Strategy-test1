# MyMomentumStrategy

This is a test strategy repository compatible with the MARKO Platform.

## Configuration

The strategy accepts the following parameters:
- `period`: Strategy period (default: 14)
- `rsi_period`: RSI calculation period (default: 14)
- `macd_fast`: MACD fast period (default: 12)
- `macd_slow`: MACD slow period (default: 26)

## Versioning

This repository follows Semantic Versioning. Releases are managed via Git tags.
